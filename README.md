# TETRA协议文档 - 第二部分：空中接口

## 概述

TETRA（Terrestrial Trunked Radio），地面集群无线电，是一种高度专业化的无线通信系统，专为公共安全机构、交通运输、铁路运营以及大型工业和商业领域设计。它利用数字时分多址（TDMA）技术，提供高效、可靠且安全的通信解决方案，满足指挥调度、实时数据交换等关键业务需求。
